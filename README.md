A smart analytics dashboard that compares product prices across marketplaces, detects price anomalies, identifies best deals, and provides AI-based buy/wait/avoid recommendations using machine learning.
This project helps users make data-driven purchasing decisions by analyzing marketplace pricing trends and predicting fair product prices.



**Features**
 Product-wise filtering
 Average, Minimum & Maximum price insights
 Interactive price distribution histogram
 Price anomaly detection using Z-score
 Deal scoring to identify best offers
 Machine Learning price prediction
 Actual vs Predicted price visualization
 Buy / Wait / Avoid recommendation
 Clickable product links
 Data preview dashboard


****Machine Learning Logic****
The system uses Random Forest Regressor to:
Learn average product pricing
Predict expected market price
Compare actual vs predicted price
Generate recommendation 


**Dashboard Sections**
**Price Intelligence** Average price, Minimum price, Maximum price
**Price Distribution** Histogram showing price variation
**Price Deviations** Detects unusually priced products
**Best Deals** Top deals based on scoring
**ML Prediction** Predicted fair market price
**Recommendations** Buy / Wait / Avoid suggestions
